README

Process:
1. Generate the truth table using the excel spreadsheet to create letters and number of your choosing.
2. Install Logic Friday application
3. Copy your truth table over and run minimization to get simplified equations.
4. Install Python 3.* (NOTE: Please install to PATH)
5. Copy over minimized Logic Friday equations (don't copy the "Minimized" heading) into input.txt
6. Open the command prompt, go into the folder where python script and input.txt are located and type: python script.py
7. The WINCUPL syntax equations will be in WINCUPL.txt which will be located in the same folder.
8. Copy and Paste into WINCUPL and you are DONE!
